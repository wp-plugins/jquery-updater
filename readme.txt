== Description ==

This plugin should only be used for development purposes, this plugin updates jQuery to the latest stable version - which is not in the WordPress core.
== Installation ==
1. Upload `jquery-updater.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
